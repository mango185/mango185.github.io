Linux命令

发表于 2020-05-28
本文字数： 2.1k 阅读时长 ≈ 2 分钟

linux常用命令：

小技巧：

1.pwd 查看当前所在位置

2.ls 查询目录中的内容

3.ctrl+L Ctrl+C（Ctrl+Z）

4.cd 移动到某个位置

5.date 查看当前服务器时间

6.ifconfig 查看自己的ip地址

7.ping 检测网络连通

8.mkdir创建目录 touch创建文件

9.rm 删除

10.cp 复制

11.mv 剪切或者改名

12.vim 编辑文件内容

13.cat 查看文件内容

14.chmod 赋权

15.netstat 查看当前端口号

16.ps -ef 查看进程号

17.su 切换为超级用户

18.Top 动态查看系统资源占有率

19.Find 根据文件名查找指定文件

20.less 查看静态日志文件

21.tail 查看动态日志文件

22.grep 过滤

23、rz上传文件 sz下载文件

24.zip 压缩文件

25.ln命令 在文件之间创建链接
